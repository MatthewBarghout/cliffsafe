# CliffSafe — Benefits Cliff Calculator

CliffSafe helps households understand and navigate the benefits cliff — the point where a raise causes a net loss in total compensation due to lost government assistance.

## Tech Stack

| Layer | Technology |
|---|---|
| Backend | FastAPI + SQLAlchemy (async) + PostgreSQL |
| Frontend | React 18 + Tailwind CSS + Recharts + Framer Motion |
| Database | PostgreSQL 15 |
| DB Admin | pgAdmin 4 |
| Containers | Docker + Docker Compose |

---

## Local Development (Docker Compose)

### Prerequisites
- Docker Desktop installed and running
- No other services on ports 3000, 5432, 5050, or 8000

### 1. Clone and configure

```bash
git clone <repo-url>
cd benefits-cliff
cp backend/.env.example backend/.env
```

### 2. Start all services

```bash
docker-compose up --build
```

This starts:
- **PostgreSQL** on `localhost:5432`
- **pgAdmin** on `http://localhost:5050` (login: `admin@cliffsafe.local` / `admin`)
- **FastAPI backend** on `http://localhost:8000`
- **React frontend** on `http://localhost:3000`

### 3. Open the app

Navigate to **http://localhost:3000** in your browser.

API docs (auto-generated by FastAPI) are at **http://localhost:8000/docs**.

## Local Development (without Docker)

### Backend

```bash
cd backend
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env            # edit DATABASE_URL to point at your local Postgres
uvicorn app.main:app --reload --port 8000
```

### Frontend

```bash
cd frontend
npm install
npm start                       # runs on http://localhost:3000
```

## API Reference

### `POST /api/calculate`
Calculate benefits cliff for a given income scenario.

**Body:**
```json
{
  "gross_income": 38000,
  "household_size": 3,
  "state": "CA",
  "employment_type": "full_time"
}
```

### `GET /api/benefits/{state}/{household_size}`
Get benefit program thresholds for a state and household size.

### `POST /api/optimize`
Get income optimization strategies to preserve benefits eligibility.

**Body:** Same as `/calculate`, plus optional `target_income`.
